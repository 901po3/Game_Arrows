# Game Plan

Arrow Puzzle 게임의 정체성, 핵심 루프, 콘텐츠·기능·밸런스 원칙을 정의한다. 기획자·프로그래머·QA가 공통으로 참고한다. 참고: 탭 이스케이프 스타일 힐링 논리 퍼즐. 아래 정체성·콘텐츠는 기획 명세와 일치하도록 유지한다.

---

## 1. 게임 정체성

- **타이틀**: Arrow Puzzle
- **장르·한 줄 요약**: 간단한 터치가 보람찬 도전이 되는 **힐링 논리 게임**. 탭 이스케이프 스타일 퍼즐.
- **핵심 루프**:
  - 그리드(미로)에 서로 다른 방향을 가리키는 화살표가 배치됨.
  - **빈 경로를 찾아** 화살표를 **올바른 순서로 탭**하면, 해당 화살표가 그리드에서 나가며 공간이 생김.
  - **앞에 다른 화살표가 경로를 막고 있으면** 그 화살표는 움직일 수 없음(막힌 화살표).
  - 화살표를 하나씩 탭해 그리드에서 내보내며, **보드를 텅 비울 때까지** 순서를 계획·실행하는 것이 목표.
  - 막힌 화살표를 탭하면 **생명 1 감소**. 레벨마다 생명 제한 있음. 막히면 **힌트**로 탭 가능한 화살표를 강조해 안내.
  - **타이머 없음**. 자신만의 속도로 계획·재도전 가능.
- **타겟 경험**:
  - 압박 없이 두뇌를 가볍게 자극, **휴식·집중**, 스트레스 해소.
  - **승리의 만족감**(보드가 비워질 때), **몰입감**(화살표를 따라가며 정신 집중).
  - **미니멀한 비주얼·부드러운 애니메이션**으로 퍼즐에만 집중.
  - 논리력·공간 지각력 훈련, 차분한 일상 루틴으로 소음에서 벗어난 경험.
- **유지 원칙**: 기획·기능 변경 시에도 위 핵심 루프(순서대로 탭해 보드 비우기)와 타겟 경험(힐링·압박 없음·만족감)을 해치지 않는다.

## 2. 콘텐츠

### 2.1 아이템

- **화살표**: 그리드에 배치되는 퍼즐 오브젝트. 방향별로 존재하며, **앞에 경로를 막는 화살표가 없을 때만** 탭해 그리드에서 내보낼 수 있음.
- **생명**: 레벨마다 제한. **막힌 화살표를 탭하면 1 감소**. 0이 되면 실패(재도전 가능).
- **힌트**: 막혔을 때 사용. **그리드에서 안전하게 빠져나갈 수 있는 화살표**가 강조 표시됨. (획득·수량 제한은 상세 기획에서 정의)
- (선택) **트로피**: 일일·월간 챌린지 완료 시 획득하는 특별 보상. (§2.4 상품·이벤트와 연동 가능)
- 기타: ([Glossary.md](../Glossary.md) 참고)

### 2.2 스테이지

- **정의**: 한 레벨 = **한 화살표 그리드(미로)**. 클리어 조건 = **그리드의 모든 화살표를 올바른 순서로 탭해 비우기**.
- **진행**: 수백 개 수준의 **핸드메이드 탭 어웨이 스타일 퍼즐**. 진행할수록 **난이도 상승**(그리드 크기·화살 배치 복잡도 등).
- **실패·재도전**: 생명 소진 시 실패. **타이머·압박 없이** 언제든 자신만의 속도로 계획을 다시 세우고 재도전 가능.
- **난이도 곡선**: 10 단위 스테이지마다 어려운 스테이지, 그 외는 상대적으로 더 쉬운 구성. (§3 참고)

### 2.3 기본 기능

- **입력**: 화살표 **탭**으로 선택·발사(그리드에서 내보내기). 빈 경로가 있는 화살표만 탭 가능.
- **판정**: 탭 가능 화살표 탭 시 제거·애니메이션; 막힌 화살표 탭 시 생명 감소.
- **힌트**: 사용 시 탭 가능한 화살표 강조.
- **UI**: 그리드, 생명, 힌트 버튼, 레벨/진행, 재시작. **타이머 UI 없음**.
- **저장/로드**: 진행도·설정 저장. 씬 전환(메인·레벨·설정 등).
- **비주얼**: 깔끔·차분한 미니멀 비주얼, 부드러운 애니메이션.

### 2.4 상품

- (추가 예정) 스토어·구매 가능 항목. 힌트·생명 등과 연동할 경우 상세 기획에서 정의.

### 2.5 광고

- (추가 예정) 연동 방식(보상형 등), 노출 시점·보상.

### 2.6 챌린지·보상

- **일일 챌린지**: 매일 플레이 가능한 챌린지.
- **월간 챌린지**: 월 단위 챌린지 완료 시 **특별 트로피** 등 보상.

## 3. 밸런스 원칙

- **난이도·보상·경제**가 일관된 곡선을 가질 것. pay-to-win 또는 과도한 불쾌감을 주지 않을 것.
- **타이머·압박 없음**: 진정한 즐거움·힐링을 위해 시간 제한을 두지 않는다.
- **난이도 배치**: 10 단위 스테이지마다 어려운 스테이지가 나오고, 10 단위가 아닌 스테이지는 상대적으로 더 쉽게 유지.
- **규칙의 단순함**: 시작은 쉽고, 마스터할수록 전략의 깊이가 느껴지도록. 한 번에 이해 가능한 규칙을 유지.
- (추가 예정 — 수치·규칙이 정해지면 여기에 요약)
