# 현재 구조 요약

프로젝트의 **주요 폴더·모듈·데이터 흐름**을 요약한다. AI가 히스토리를 잃었을 때 "지금 구조가 어떤지" 복원하기 위함.

---

## 규칙

- **구조가 바뀌면 이 문서를 갱신한다.** (폴더 추가·삭제, 주요 매니저/시스템 변경, 데이터 흐름 변경 시.)
- 프로그래머·코드 리뷰어는 새 시스템 추가·리팩터 시 이 문서를 참고하고, 변경 후 여기를 업데이트한다.

---

## 현재 폴더·모듈 (예시)

- **Assets/Scripts/**: (기능별 하위 폴더: Items, Stage 등 — 실제 구조에 맞게 채움)
- **Assets/Features/** (사용 시): 기능별 Scripts, Prefabs, Art 등
- **docs/plan/**: GamePlan, 기획 명세, CurrentStructure
- **docs/changelog/**: 작업 요약(브랜치·날짜별)

(구현이 진행되면 위 목록을 실제 구조로 채우고, 주요 매니저·이벤트 흐름을 짧게 적는다.)
